Elementos

TAGS de HTML5

!DOCTYPE html

Es una instrucción especial que va al inicio de nuestro documento HTML y que permite al navegador entender qué versión de HTML estamos utilizando. Esta información determinará la manera en la que el navegador procesará el documento.

html

Representa la raiz de un documento HTML. El resto de elementos descienden de este elemento.

head

Provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

title

Es un tag de código HTML que le permite darle un título a una página web.

body

Es el contenedor para todos los aspectos representables de un documento. Este es el lugar donde los autores arrojan todos los elementos y contenidos que sus visitantes alcanzarán.

h1

Es el elemento HTML utilizado, de manera habitual, para identificar la cabecera más importante en una página web.

p

El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.
